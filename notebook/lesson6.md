### what is AWS
> AWS stands for Amazon Web Services, which is Amazon's comprehensive cloud computing platform. It's one of the world's largest and most widely-used cloud service providers.

#### Core Services:
1. Compute - Virtual servers (EC2), serverless computing (Lambda), containers
2. Storage - Object storage (S3), block storage (EBS), file systems
3. Databases - Relational (RDS), NoSQL (DynamoDB), data warehousing (Redshift)
4. Networking - Virtual private clouds (VPC), content delivery (CloudFront), load balancing
